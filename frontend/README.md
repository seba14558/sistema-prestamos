# Préstamos Frontend

Este proyecto es un frontend en React para la gestión de préstamos, con autenticación, paneles diferenciados para administrador y cobrador, carga y edición de clientes y préstamos, visualización de vencimientos, morosos y recaudación. Consume la API en http://localhost:3001.

## Funcionalidades principales
- Login de usuario (admin/cobrador)
- Panel de administrador
- Panel de cobrador
- Gestión de clientes
- Gestión de préstamos (alta, edición, listado)
- Visualización de vencimientos y morosos
- Panel de recaudación

## Tecnologías
- React + TypeScript
- Material UI
- React Router
- Axios

## Instalación
1. Instalar dependencias:
   ```sh
   npm install
   ```
2. Iniciar la app:
   ```sh
   npm start
   ```

## Notas
- Reemplazar los placeholders de UI y lógica según necesidades reales.
- La API debe estar corriendo en http://localhost:3001.
